このサイトについて

「あにあに遊び場」は、作品をサクッと公開して反応をもらうための場所。まずはカードゲームのデモから。全部静的HTMLなので無料で公開できます。

技術メモ

  • GitHub Pages（無料）でホスティング
  • 純HTML/CSSのみ（ビルド不要）
  • OGP・アイコン済／モバイル最適化

連絡

改善アイデア・不具合報告などは、いま話しているチャットに送ってもらえればOK。公開用メールやフォームも後で追加できます。
